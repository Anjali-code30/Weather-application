<h1> Weather-application </h1>
A simple Weather Application that allows user to fetch data in real time,
by city name.
<br>
<h2>Features</h2>
<ul>
<li>Search weather by city name</li>
<li>Displays live temperature in °C</li>
<li>Shows humidity and feels like temperature</li>
<li>Real-time data using OpenWeatherMap API</li>
</ul>

<h2>Tech Stack</h2>
<ul>
<li>HTML</li>
<li>CSS (Tailwind CSS)</li>
<li>JavaScript</li>
<li>OpenWeatherMap API</li>
</ul>
